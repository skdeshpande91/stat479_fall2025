Linear Regression

Adjusted plus/minus attempts to account for the other 9 players.

Formally, it assumes that there is some number \(\alpha_{p}\) associated with each player \(p\) and an average home-court advantage \(\alpha_{0}\) such that \[ Y_{i} = \alpha_{0} + \alpha_{h_{i1}} + \cdots + \alpha_{h_{i5}} - \alpha_{a_{i1}} - \cdots - \alpha_{a_{i5}} + \epsilon_{i}, \] where the error term is assumed to have mean 0.

How can we estimate these using data?

One approach is least squares.

Remember that big design matrix \(X\) from earlier? It turns out we can re-formulate this problem as \[ \lVert Y - \tilde{X} \boldsymbol{\alpha} \rVert_{2}^{2} \]
